About

I'm Walt Parkman, age 70 years. I waited patiently for AI while in 15 software developer positions. I knew it would come someday.

Today 03/28/2025, Codeium Windsurf, with the Cascade coding agent backed by the latest Google Gemini 2.5 LLM model is a powerful AI system. One person can now rapidly enhance and test massive codebases. A popular open source one is Odoo ERP.

On my home page, I confidently declare that I do Odoo Vibe Coding. The term "vibe coding" was introduced by computer scientist Andrej Karpathy (co-founder of OpenAI and former AI lead at Tesla) in early 2025. Vibe Coding relies totally on Large Language Models (LLMs). Instead of manually writing code, the programmer uses natural language descriptions or conversational prompts (like voice commands) to have the AI generate the code.

This really works now. New models keep coming out and exponential improvement is happening. Vibe Coding will soon be the customary way of coding. Robust and secure code from the LLM will be better than human-generated code.

If you feel that vibe, then welcome to the tribe. The tribe still does code reviews, but only to admire the work of the AI.
